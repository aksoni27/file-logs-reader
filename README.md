
## Running the app

```bash
# development
$ node server/server.js

# Server will be running on port - 3000 

# To view raw data run endpoint -
$ http://localhost:3000/logs?start=2020-01-01T00:02:25.365Z&end=2020-08-01T00:04:58.279Z
# here start is startime and end is endtime passed as query params to view logs for those particular timeframe
```
